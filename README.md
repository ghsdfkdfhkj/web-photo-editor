# 웹 포토 에디터

간단한 웹 기반 이미지 편집 도구입니다. Fabric.js를 사용하여 구현되었습니다.

## 기능

- 이미지 업로드
- 밝기 조절
- 이미지 회전
- 텍스트 추가
- 편집된 이미지 저장

## 사용 방법

1. 웹 브라우저에서 index.html 파일을 엽니다
2. "파일 선택" 버튼을 클릭하여 이미지를 업로드합니다
3. 도구 모음의 버튼들을 사용하여 이미지를 편집합니다
4. "이미지 저장" 버튼을 클릭하여 편집된 이미지를 다운로드합니다

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Fabric.js
